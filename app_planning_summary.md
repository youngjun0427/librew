# Coffee Brew Log — 앱 기획 요약 (v2)

커피 핸드드립 취미 유저를 위한 앱. 1인 프론트엔드 개발, 이직 포트폴리오.

---

## UI 방향

**하단 탭바 없음.** 단일 스크롤형 메인 홈 화면 + 섹션별 관리 화면 진입 방식.

메인 홈 구성 (이미지 참고):

- 상단: 장비 세팅 섹션 (현재 사용 중인 그라인더/케틀/드리퍼/저울 카드)
- 중단: 레시피 관리 섹션 (레시피 카드 리스트)
- 하단 고정: "브루잉 시작" 버튼 (눈에 띄는 강조색)

---

## MVP 범위

**MVP 포함 (4가지):**

1. 원두 관리
2. 장비 관리
3. 레시피 관리
4. 추출 기능 (브루잉 플로우 전체)

**MVP 이후 (Post-MVP):**

- Gemini AI 개선 제안 (추출 기록 기반)
- 레시피 공유 (공개 링크)
- 웹 사이드바 레이아웃

---

## 기능 명세

### 1. 장비 관리

- 장비 등록: 종류(그라인더/케틀/드리퍼/저울/기타), 브랜드, 모델명, 종류별 스펙, 메모
- 현재 세팅 지정: 장비별 `isInCurrentSetup` 플래그로 홈 화면에 표시
- 장비 조회 / 수정 / 삭제

### 2. 원두 관리

- 원두 등록: 원산지, 로스터리, 품종, 로스팅 날짜, 구매일, 가격, 용량(g)
- 잔여 용량 자동 차감: 추출 완료 시 사용량만큼 자동 차감
- 구매 이력 및 누적 지출 조회

### 3. 레시피 관리

- 레시피 생성: 분쇄도, 물 온도, 원두량(g), 물량(g), 필터, 추출 방식
- 추출 단계(steps) 등록: 차수별 물양(ml), 붓는 시간(초), 대기 시간(초), 푸어링 방식, 팁
- 레시피 조회 / 수정 / 삭제

### 4. 추출 플로우 (브루잉)

아래 단계 순서로 진행:

| 단계 | 화면        | 내용                                                                    |
| ---- | ----------- | ----------------------------------------------------------------------- |
| 1    | 장비 확인   | 현재 세팅된 장비 확인, 변경 가능                                        |
| 2    | 레시피 선택 | 저장된 레시피 목록 / 새 레시피 추가 후 재진입                           |
| 3    | 원두 선택   | 잔여 용량 기준 자동 그램수 보정                                         |
| 4    | 카운트다운  | 3초 (저울/케틀 준비용)                                                  |
| 5    | 추출 진행   | 차수별 물양·시간 안내, 팁, 푸어링 방식 추천                             |
| 6    | 추출 완료   | 총 소요 시간, 요약                                                      |
| 7    | 평가        | 관능 평가(산미/쓴맛/바디/향/총점 1-5) + 자유 메모 + 다음 추출 개선 지표 |

추출 완료 시 자동 처리:

- brewLogs에 기록 저장
- 원두 잔여 용량 차감

---

## 핵심 유저 플로우

```
[홈] 브루잉 시작 버튼 →
  [장비 확인] → [레시피 선택] → [원두 선택] →
  [3초 카운트다운] → [단계별 추출 진행] → [완료] →
  [관능 평가 + 메모] → [홈으로]
```

## 화면 목록

| 화면                 | 주요 구성                                       | 진입 방식           |
| -------------------- | ----------------------------------------------- | ------------------- |
| 메인 홈              | 장비 세팅 섹션 + 레시피 섹션 + 브루잉 시작 버튼 | 앱 진입             |
| 장비 관리            | 장비 목록 (종류별 아이콘, 스펙 표시)            | 홈 > 장비 섹션 편집 |
| 장비 등록/수정       | 종류 선택 + 스펙 입력 폼                        | 장비 관리           |
| 레시피 관리          | 레시피 카드 리스트 (파라미터 요약 표시)         | 홈 > 레시피 섹션    |
| 레시피 생성/수정     | 파라미터 + 단계 입력 폼                         | 레시피 관리         |
| 레시피 상세          | 전체 파라미터, 단계 미리보기                    | 레시피 관리         |
| 원두 관리            | 목록, 잔여 용량 바, 구매 이력                   | 홈 > 메뉴           |
| 원두 등록/수정       | 원두 정보 입력 폼                               | 원두 관리           |
| 추출 기록            | 추출 일지 목록                                  | 홈 > 메뉴           |
| 기록 상세            | 레시피·원두·관능 평가·메모                      | 추출 기록           |
| 브루잉 - 장비 확인   | 현재 장비 세팅 표시/변경                        | 브루잉 시작         |
| 브루잉 - 레시피 선택 | 저장 레시피 목록 카드                           | 장비 확인 완료      |
| 브루잉 - 원두 선택   | 원두 목록, 잔여량, 자동 보정                    | 레시피 선택 완료    |
| 브루잉 - 카운트다운  | 3초 카운트다운 UI                               | 원두 선택 완료      |
| 브루잉 - 추출 진행   | 차수별 물양/타이머/팁/푸어링 추천               | 카운트다운 완료     |
| 브루잉 - 완료        | 총 시간, 요약                                   | 마지막 차수 완료    |
| 브루잉 - 평가        | 관능 평가 + 메모 + 개선 지표                    | 완료 화면           |
| 공유 링크 (Post-MVP) | 레시피 미리보기, 가져오기                       | 딥링크              |
